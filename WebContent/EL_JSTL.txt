 - XML라이브러리
  -> XML 문서 처리

 EL - 내장객체(p513~)
 * pageScope, requestScope, sessionScope, applicationScope
   => JSP의 내장객체를 사용가능 (영역에 있는 속성을 공유가능)
 * param, paramValues
   => 파라미터 정보를 저장하기 위한 객체 / paramValues(동일한 이름의 파라미터정보를 배열의 형태로 저장)
 * header, headerValues
   => Header 정보를 저장하는 객체  // 배열로 저장하는 객체  
 * cookie 
   => JSP 에서 사용하던 Cookie 객체
 * initParam 
   => 컨텍스트(프로젝트) 초기화 파라미터정보를 저장하는 객체
 * pageContext
   => pageContext(프로젝트) 객체를 참조할때 사용
